El comando mágico de toda la vida para arrancarlo en la terminal (por si alguna vez necesitas reiniciarlo por tu cuenta) es:

bash
npm run dev

👉 Desde la PC: http://localhost:5173/ 
👉 Desde tu teléfono con Wi-Fi U otra PC (red local): http://192.168.1.103:5173/

_Normalmente, el flujo completo para guardar y aplicar todo es:
								Guardar los archivos en el editor (Ctrl + S).
Compilar el proyecto web (esto genera la carpeta dist o www):
							     powershell:
									npm run build
Sincronizar esos archivos con la carpeta de Android:
						    powershell:
							       npx cap sync
Si estás usando el servidor de desarrollo (npm run dev) y tienes el teléfono conectado, los cambios de código se suelen aplicar instantáneamente (Hot Reload) sin comandos adicionales, siempre y cuando el servidor esté corriendo.
